# 物品特征

## 格式

* 同一个物品的不同特征匹配，用 , 分开
* 多个物品特征用 ; 分开

## 示例

```text
material:DIAMOND,amount:64 ; material:EMERALD,amount:32
```

## 特征

* 材质: Material
* 损伤值: Data
* 数量: Amount
* 名称: Name
* 描述: Lore
* 头颅材质: Texture
* ModelData
